# Team Roles

The team has 3 members. Responsibilities follow the project implementation plan.

## Developer A

Primary area:

- Platform foundation
- API Gateway
- Discovery Server
- Identity Service
- Security
- CI/CD

Early milestones:

- Milestone 0: repository and team workflow setup
- Milestone 1: platform foundation
- Milestone 2: identity and access management

## Developer B

Primary area:

- Department Service
- Employee Service

Early milestones:

- Milestone 3: organization and employee management

## Developer C

Primary area:

- Attendance Service
- Reporting Service
- Kafka consumers
- Notification Service

Early milestones:

- Milestone 4: attendance
- Milestone 5: reporting and notifications

## Shared Responsibilities

All developers are responsible for:

- Writing and reviewing specs
- Reviewing pull requests
- Keeping tests meaningful
- Respecting service boundaries
- Updating documentation when behavior changes
